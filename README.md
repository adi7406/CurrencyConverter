💱 Currency Converter

A simple and interactive Currency Converter web application built using HTML, CSS, and JavaScript.
This application allows users to convert one currency into another using real-time exchange rates fetched from an API.

The project also dynamically updates country flags and currency options, making the interface more user-friendly.

🚀 Features

🌍 Convert between multiple international currencies

📡 Fetch live exchange rates using an API

🏳️ Automatic country flag updates when selecting currencies

⚡ Instant conversion results

🎨 Clean and responsive UI

🔄 Swap-style currency selection interface

The application fetches exchange rate data from an online API inside the JavaScript logic.

⚙️ How It Works

The user enters an amount to convert.

The user selects the source currency and target currency.

The app fetches live exchange rates from the currency API.

The result is calculated and displayed instantly.

The conversion formula used is:

Converted Amount = Amount × Exchange Rate
